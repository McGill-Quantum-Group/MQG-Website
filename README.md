# The McGill Quantum Group (MQG) Website

Welcome! This is where all the website's source code lives. If you would like to run it yourself (for whatever reason), feel free to clone this repo and run npm install to get all the directories up and running.  

If you encounter a bug, please let us know in the issues tab, or on the MQG Discord (https://discord.gg/KY4e9BrQNe).  

We hope you enjoy the website!
